Welcome to COP-CW - Copernicus Coastal Web Viewer.


This is an Open Source Web platform that aims to integrate multiple geospatial data sources, specially from Copernicus services, that are of interest for the study of the Italian coasts. It allows the parametric visualization of multiple geospatial layers, simple data analysis, and experiment sharing.

It uses the Mediator/Wrapper architecture, as a data integration strategy, to consume the data directly from the services instead of replicating it. It currently includes data from Copernicus Marine Environment Monitoring Service (CMEMS), Copernicus Land Monitoring Service (CLMS), Copernicus Atmosphere Monitoring Service (CAMS), Italian Geoportal, WorldPop, but more services are being incrementally included.

To use the platform, follow this 5-step workflow, which will be further explained in each of the following sections of the tutorial.

Step 1: Context Definition


First, in order to filter and select between all the data that is currently available, it is necesary to set a spatiotemporal context. This context is the combination of a range of dates (1) and an Area of Interest (2).
